## Accessing SOAR Templates

### Step 1: Open the menu

1. Tap the **three horizontal bars** (â˜°) on the top-left corner of the app
    
2. Select **SOAR Template**
    

---

### Step 2: Enter authorization password

1. Enter the required **password**
    
2. Once validated, you will be granted access to the template menu
    

---

## Selecting a Template Type

After logging in, you will see template options for:

- **Sales Template**
    
- **Expense Template**
    
- **Requisition Template**
    
- **Inventory Template**
    

Select the template you want to manage.

---

## Editing a Template

### Step 3: Open the template

1. Tap the template type (example: **Sales Template**)
    
2. You will be directed to the template editor page
    

---

### Step 4: Manage template items

Inside the template editor, authorized users can:

- **Sort items**
    
    - Rearrange the order in which items appear
        
- **Add items**
    
    - Select items to include in the template
        

> ðŸ“Œ **Note**  
> Only items that already exist in the **system product list** can be added to a template.

---

### Step 5: Review template changes

Before saving:

- Check the item order
    
- Confirm all required items are included
    
- Make sure no incorrect items were added
    

---

### Step 6: Save the template

- Click **Save** to apply the changes
    
- The updated template will now be used in:
    
    - Sales
        
    - Expenses
        
    - Requisition
        
    - Inventory (depending on the template edited)
        

---

## Important Rules and Reminders

- Only **authorized users** can edit templates
    
- Templates affect what store crews see during transactions
    
- Items must exist in the **product list** before they can be added
    
- Changes apply immediately after saving
    

> âš ï¸ **Warning**  
> Incorrect template changes may affect daily operations.  
> Always review templates carefully before saving.
